# Create a shopping cart

The project is create shopping cart 

Home page will have 

- search on top
- filer 
- infinite scroll
- carousel for trending items
- recommended items list 
- Add item to cart
- view an item
- product  images
- product detail
- select no of items
- add rating

Cart page:

- List of all items in the cart
- suggested item to add in the cart
- total amount
- discount applied on each product
- total discount 
- apply coupon for additional discount
- checkout
- add address and details / validate the details


Other features:

- Dark mode/ Light mode
- Sidebar to see different products 
- Loader animation
- carousel animation
- Product should appear cool

Product schema 

[{
    id: number,
    name: string,
    actualPrice: number,
    description: string,
    images: sting[],
    discountPercentage: number,
    tags: string[],
    rating: number[]
}]
