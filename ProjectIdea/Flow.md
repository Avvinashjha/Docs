# Project Flow

Idea:

- Create a platform where user can create pipeline
- drag and drop different types of data store
- when user connects two data store then an edge will form and edge can be of different type
- data can be batched and streamed

datastores supported:

- kafka
- druid
- hadoop
- mysql
- postgres
- mongodb
- sqlite
- file
- json
- csv
- can be expandable for more

Features:

- User can create a flow
  - User can select datastores
  - user can created edges between different data stores
  - user can add etl
  - user can see different lags
  - 
- use can create one time data transfer
- user can replicate data transfer and it will run as given time interval
